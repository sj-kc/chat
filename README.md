# Chat app.

* Nodejs.
* Express.
* Socket-io.


### Clone the repo

```bash
git clone https://github.com/sungkimcastro/chat.git
```

### Install dependencies

```bash
yarn install
```

### Start a local server to preview the site

```bash
yarn dev
```
